# Symboli Prelude

> _Eclipse first, the rest nowhere._

Symboli Prelude is the core utility for some plugins.

## Build

Open `symboli_prelude.sln` and build it.
Of course you can use `msbuild` :

```bat
MSBuild -p:Configuration=Release
```
